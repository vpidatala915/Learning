1. The "Source of Truth" Libraries
MDN Web Docs Glossary: This is the gold standard for Web and Networking terms (HTTP, DNS, Host, CORS). If you hear a term like "Host Authority" or "Payload," look it up here first.

CNCF Cloud Native Glossary: This is the best place for Infrastructure and Modern Ops terms (Kubernetes, Microservices, Latency, Sidecars). It’s specifically designed to be "jargon-free" for both tech and business people.

Microsoft Azure Cloud Dictionary: Even if you don't use Azure, this is an excellent high-level list for Cloud Architecture (IaaS, PaaS, Scalability, High Availability).
